# Roadmap

## Fase 1 — Core Engine ✅
AgentEngine, EventBus, MemoryEngine, ToolEngine, Workflow básico, testes iniciais.

## Fase 2 — UI ✅
DashboardComponent, Sidebar, AgentList, LogViewer, CreateAgentDialog, integração EventBus.

## Fase 3 — Governance + Sandbox ✅
GovernanceEngine (compliance, hierarquia, drift, trust score), Sandbox (PolicyEngine, Workspace).

## Fase 4 — Change Management ✅
Snapshots, DiffEngine, ChangeManager (Pending→Approved/Rejected), RollbackManager, RecoveryEngine, Emergency Stop.

## Fase 5 — Workflow Engine ✅
Orquestração CEO→Managers→Teams, decomposição de objetivos, auto-complete, pause/resume.

## Fase 6 — AI Integration ✅
PlannerEngine, ModelRouter, AgentProfiles, ReasoningEngine, ContextEngine, PromptEngine, ObjectiveEngine, CostMonitor, UI panels.

## Fase 7 — Multi-Agent Collaboration 🔜
- Agentes conversam entre si
- Debate e consenso entre agentes
- Atribuição dinâmica de tarefas
- Resolução de conflitos

## Fase 8 — Plugin Ecosystem & SDK 🔜
- Plugin SDK oficial
- Marketplace de plugins
- Providers de LLM como plugin
- Ferramentas personalizadas

## Fase 9 — Distributed AgentOS 🔜
- Agentes rodando em múltiplas máquinas
- Comunicação via rede
- Workflows distribuídos
- Orquestração cross-instância

## Futuro (ideias)
- GitEngine: integração com git
- ResearchEngine: pesquisa autônoma
- SemanticSearch: busca semântica no código
- KnowledgeGraph: grafo de conhecimento do projeto
- ASTAnalysis: análise de código fonte
- Auto Coding: geração supervisionada de código
