JWT Authentication Implementation

O JSON Web Token (JWT) e um padrao para autenticacao stateless.

Estrutura do JWT:
- Header: algoritmo (HS256, RS256) e tipo do token
- Payload: claims como sub, iat, exp, role
- Signature: assinatura do header + payload

Implementacao em C++:

Fluxo de autenticacao:
1. Usuario faz login com usuario/senha
2. Servidor valida credenciais e gera JWT
3. Cliente armazena o token (localStorage/cookies)
4. Cliente envia token no header Authorization: Bearer <token>
5. Servidor valida o token em cada requisicao protegida
